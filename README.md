## Project overview

Predict Mundial 2026 allows users to simulate the entire World Cup tournament, including:

- Group stage with official FIFA rules
- Best third-place team qualification
- Playoff and repechage integration
- Full knockout bracket up to the final
- Automatic prediction modes
- Multi-language support (ES / EN)
